We are in the process of further refining the Geonosis™ Database Engine by creating some ColdFusion CFC Objects to help in the process of manipulating Geonosis™ Objects.

Just to recap, Geonosis™ in a Web based AJAX-powered Content Management System and Application Generator that uses the Geonosis™ Database Engine. The Geonosis™ Database Engine turns SQL Server 2000 or SQL Server 2005 into an Object-Oriented Networked Database that allows for ad-hoc data to be stored and manipulated with complete ease without the need to code or know SQL.
