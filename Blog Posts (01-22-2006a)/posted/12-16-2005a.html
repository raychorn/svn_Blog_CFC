Geonosis Version 1.0 - Adding the User Password Manager.

The User Password Manager provides Geonosis with the automatic ability to ensure all users are provided with an email that requires the user to "activate" a new user account prior to allowing a new user to log-in.

The really cool thing about Geonosis is that any module or programmatic function that is part of Geonosis can be made accessible to users of Geonosis and can therefore be made part of any web app that Geonosis can generate. This means any web app that Geonosis generates will be fully able to manage users using the same technique that Geonosis uses to manage users.

Got a need to deploy a web based user management system ? Well sure you do because any web site that need to manage users or subscribers will need to manage users. No more need to code this functionality for every single site one needs to deploy. No more need to code this ever again. Simply use Geonosis to generate all your web apps that need to manage users.

Stay tuned for more posts about Geonosis and this exciting new way to create and manage web based content.
