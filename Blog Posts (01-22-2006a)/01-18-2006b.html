Those who can code products do. Those who cannot, code open source.

The Open Source concept was created by those who lack the abilities or the desire to spend the time working on real products because coding real products take time and energy and desire and committment. Mainly it takes creativity and resourcefulness. Open Source advocates are not willing to do the hard work required to make real products come to fruition. Open Source provides a way for slothful Bart Simpson types to try to gain some fame they would otherwise be denied. Open Source fosters the desire to lay claim on code others write simply because they feel all code related to open source becomes open source.
